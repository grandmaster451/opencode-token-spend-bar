# Changelog

All notable changes to this project will be documented in this file.

## [1.0.0] - 2026-04-13

### Added

- Initial release of opencode-token-spend-bar
- Session-only token spending widget in `session_prompt_right` slot
- Current calendar month aggregation with automatic cache rebuild on month change
- MiniMax API cost tracking
- OpenCode Go cost tracking
- ChatGPT Plus usage display (token counts, no currency)
- Usage-only fallback when cost metadata is unavailable
- Narrow terminal fallback for constrained widths
- Package examples directory with configuration reference
